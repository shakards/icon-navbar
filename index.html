Navbar with Icon Tutorial

This tutorial demonstrates how to create a simple HTML navigation bar with icons.

What is a Navbar with Icons?

A navbar with icons is a navigation bar that includes both text and icons for each link. The icons can help users quickly identify the sections or pages they are accessing.

Why Use Icons in a Navbar?

  • Enhanced Visual Appeal: Icons make the navbar more engaging and user-friendly.
  • Quick Navigation: Icons help users quickly understand where each link leads.
  • Modern Look: Adding icons gives the navbar a more contemporary, sleek design.
        <!DOCTYPE html>
        <html lang="en">
        <headmeta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Navbar with Icon</title>
          <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css">
          <style>
            .navbar1 {
              display: flex;
              align-items: center;
              background-color: #333;
              padding: 10px 20px;
            }
            .navbar1 .logo1 {
              font-size: 20px;
              font-weight: bold;
              color: white;
              margin-right: auto;
            }
            .navbar1 a {
              color: white;
              text-decoration: none;
              padding: 10px 15px;
              margin: 0 5px;
              display: flex;
              align-items: center;
            }
            .navbar1 a:hover {
              background-color: #575757;
              border-radius: 5px;
            }
            .navbar1 i {
              margin-right: 8px;
            }
          </style>
        </head>
        <body>
          <div class="navbar1">
            <div class="logo1">MyWebsite</div>
            <a href="#home"><i class="fas fa-home"></i>Home</a>
            <a href="#about"><i class="fas fa-info-circle"></i>About</a>
            <a href="#services"><i class="fas fa-cogs"></i>Services</a>
            <a href="#contact"><i class="fas fa-envelope"></i>Contact</a>
          </div>
        </body>
        </html
